## Axes

| Axis | Index | Notes |
|------|-------|-------|
| Left Stick X | 0 | -1 left, +1 right |
| Left Stick Y | 1 | -1 up, +1 down |
| Right Stick X | 2 | |
| Right Stick Y | 3 | |
| L2 (LT) | 4 | 0 to 1 (trigger) |
| R2 (RT) | 5 | 0 to 1 (trigger) |

## Current Usage

- **L2 (Axis 4)**: Hold + D-Pad = Select Character
- **R2 (Axis 5)**: Hold + D-Pad = Select Level
- **L1 (Button 9)**: Reset Physics to Defaults
- **D-Pad**: Navigation (when in selection mode)
- **Cross (Button 0)**: Jump
